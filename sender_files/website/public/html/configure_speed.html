Configure Speed Help, por favor.


This is the configure speed page, where you're able to set how fast you want the packets to be sent out. Specifically, you set how many packets per seconds you want to be sent out. There's also an informational progress bar that tells you what the calculated bandwidth will be. Keep in mind that this bar uses both the pps that is entered and the currently configured packet on the C side, so if there's no packet on the C side, then it will just display 0.

The maximum line rate is around 100 Mbps, however, due to the (lack of) power in a Raspberry Pi, in reality you will probably be getting far less than that with smaller packets (down to about 5 Mbps with 64 byte packets). If you want to increase the actual bandwidth, try increasing the size of the packets.
The Pi currently does not have a packet configured, and as such the bandwidth calculation will not work until you have configured a packet for the Pi to send.
